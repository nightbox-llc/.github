# Nightbox LLC

> Cancer is a reversion. We block it.

Wyoming-incorporated, Los Angeles-operating computational biology and gene therapy company.

**Lead asset:** NKG2D-LIF6, a 2,123 bp chimeric receptor-effector AAV9 gene therapy fusing the human NKG2D stress-ligand recognition domain with the elephant LIF6 mitochondrial pro-apoptotic effector. Encodes the elephant cancer-resistance mechanism behind Peto''s paradox into a single tumor-restricted construct.
