## Three product pillars

### 🧬 Pillar 1 — NKG2D-LIF6 chimeric gene therapy
First-in-class single-construct AAV9 gene therapy for solid tumor oncology.
2,123 bp construct fusing the human NKG2D stress-ligand recognition domain
with the elephant LIF6 mitochondrial pro-apoptotic effector — encodes the
cancer-resistance mechanism behind Peto's paradox into a tumor-restricted
construct. **In silico complete · Q3-Q4 2026 in vivo · 2027 first-in-human.**
Provisional patent filed 2026-04-29.

[Preprint](https://nightboxllc.com/preprint) · [Methods](https://nightboxllc.com/research) · [3D viewer](https://nightboxllc.com/3d) · [chimera-design-notes](https://github.com/nightbox-llc/chimera-design-notes) (CC BY 4.0)

### ⚡ Pillar 2 — UniGPU vendor-neutral runtime
GPU-vendor-neutral compute runtime in Rust. **47.31 TFLOP/s WMMA sustained
on consumer AMD RX 7700.** 1.42 PETAFLOP / 30 seconds. Benchmark autonomously
executed by Claude AI assistant in agent session — raw logs preserved. Targets
the federal mandate to run AI inference on US-origin hardware regardless of
vendor stack.

### 🦆 Pillar 3 — SilverDuck (Кряк) AI agent stack
Local-first multi-agent AI stack with persistent memory, vision, OSINT.
Powered by proprietary **Nightbox NB-R14B** LLM (Apache 2.0). **Hybrid
post-quantum encrypted cloud-LLM handoff** (X25519 + ML-KEM-1024 + AES-256-GCM,
FIPS 203 + FIPS 197). Sensitive payload stays sealed in transit to commercial
cloud LLMs (Claude, Gemini, GPT, Grok). Local agent parses cloud-LLM output
through verified envelope; never leaks plaintext to network observers.
